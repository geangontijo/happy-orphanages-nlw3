# happy-orphanages-nlw3

- Projeto fullstack de controle de orfanatos

# O que eu aprendi
- Utilizar React.js
- Utilizar React Native
- Utilizar Node.js
- Utilizar Yarn
- Utilizar Typescript
- Utilizar Expo
- Utilizar Linux com ambiente WSL (o projeto foi desenvolvido nesse esquema)
- Criar mapas com o Leaflet.js (https://leafletjs.com/)
- TypeOrm do Node.js

Como instalar: 

- Backend
```
cd backend
```

```
yarn install
```

```
yarn dev
```

- Web

```
yarn install
```

```
yarn start
```

- Mobile

```
yarn install
```

```
yarn start
```
